🎮 HOW TO USE:

1. **Manual Update**
   python main.py --update-database
   
   Output: "✨ CẬP NHẬT HOÀN TẤT - Tổng: 28 payloads mới"

2. **Enable Daily Auto-Updates**
   python main.py --schedule-updates
   
   - Runs hàng ngày lúc 2:00 AM
   - Chạy background (non-blocking)
   - Press Ctrl+C to stop

3. **Check Status**
   python main.py --update-status
   
   Shows:
   • Scheduler status (RUNNING/STOPPED)
   • Total payloads (28)
   • Last updated timestamp
   • Data sources

4. **View WAF Bypass Techniques**
   python main.py --show-waf-bypass
   
   Shows 15+ techniques:
   • URL Encoding, Hex Encoding, Double encoding
   • MySQL Comments, Inline comments
   • Case Variation
   • Whitespace Variation, Buffer Overflow

5. **View Payload Database**
   python main.py --show-payloads
   
   Shows database statistics
